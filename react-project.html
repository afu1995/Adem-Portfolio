Learning React myself with focus on Routes

For this assignment we were giving a task to find an exciting feature in React and I chose Routes. Be aware of that we were given this assignment with almost zero experience with React and that is what makes this exciting. It was all about learning how React works by experiencing it yourself and to add a feature. The code I made is nothing special or complicated it is just the experience that I find so useful.

I used Routes in React where the URL changes and the Routes also nest. Nothing special but a great experience where I learned that when you put the time and effort into something it will work out some way or another.
