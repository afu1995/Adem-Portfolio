Creating a web shop with an Administrator and a products page

This was my first big PHP assignment and it was individual. We were given a task to create a web shop with following requirements:
- Two different user. An admin and a normal user.
- The system must use session to know about the logged in user. The admin must have CRUD functions.
- A registration page where the email must be used as a username for login.
- Validation form fields with Regular Expression

The whole project was to learn the basics of PHP. The design was not a focus point. Focus was the functionality of the site. I made a web shop for bikes completing all the requirements.
