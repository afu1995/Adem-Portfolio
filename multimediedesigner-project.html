Designing and weveloping my first big project for a client

We had for the first time a big project for a client. Low-Fi wished for a new visual identity. Therefor we deisgned them a new logo, a video that represents Low-Fi and a website made with HTML/CSS and JavaScript.

The code itself was a challenge. We had very little experience with HTML/CSS but with JavaScript we really did not have any. We had to learn and do they functionalities we wished for ourselves. Wish for me is the best way to learn.

If I could make anything different in the design now it would be the whole design itself. We tried to make a cozy expression but instead for me it seems like a very dark and creepy website. But back then it was great!
